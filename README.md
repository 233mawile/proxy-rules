# proxy-rules

这是一个个人使用的 Clash 规则仓库，实际建立在下面两个项目之上：

- [233mawile/subtrans](https://github.com/233mawile/subtrans)
- [Loyalsoldier/clash-rules](https://github.com/Loyalsoldier/clash-rules)

其中广告拦截用的 `reject` 规则源已切换为
[217heidai/adblockfilters](https://github.com/217heidai/adblockfilters) 提供的
`adblockmihomo.yaml`。

仓库里主要保存自定义规则列表、`subtrans` 处理脚本、打包后的发布产物，以及最终生成的订阅链接。
